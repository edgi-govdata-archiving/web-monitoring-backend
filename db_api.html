API Documentation¶

Client(email, password[, url]) Communicate with web-monitoring-db via its REST API.
Client.from_env() Instantiate a Client by obtaining its authentication info from these environment variables:
Client.list_pages(*[, chunk, chunk_size, …]) List all Pages, optionally filtered by search criteria.
Client.get_page(page_id) Lookup a specific Page by ID.
Client.list_versions(*[, page_id, chunk, …]) List Versions, optionally filtered by serach criteria, including Page.
Client.get_version(version_id) Lookup a specific Version by ID.
Client.add_version(*, page_id, capture_time, …) Submit one new Version.
Client.add_versions(versions, *[, update, …]) Submit versions in bulk for importing into web-monitoring-db.
Client.list_changes(page_id) List Changes between two Versions on a Page.
Client.get_change(*, page_id, to_version_id) Get a Changes between two Versions.
Client.list_annotations(*, page_id, …[, …]) List Annotations for a Change between two Versions.
Client.get_annotation(*, annotation_id, …) Get a specific Annontation.
Client.add_annotation(*, annotation, …[, …]) Submit updated annotations for a change between versions.
